## 🚀 三步启动 / 3-Step Quick Start / 三步啟動

```bash
# 1. 下载源码
git clone https://github.com/deepseek7878/texas-holdem-engine.git
cd texas-holdem-engine

# 2. 安装依赖
npm install   # Node.js
# 或
pip install -r requirements.txt  # Python

# 3. 运行示例
npm run demo  # 运行Demo
# 或
python demo.py
```
